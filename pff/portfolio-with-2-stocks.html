3.1 Equally-weighted portfolio

In an equally-weighted portfolio with 2 stocks, each stock has the same weight, i.e., 50%. The number of stocks corresponds to the number of columns in the dataframe. shape returns a list with two elements, the number of rows and the number of columns.

3.1.1 Portfolio returns

If we assume constant and equal weights, the portfolio return is:

\[\begin{equation} \begin{split} R_{P, t} &= \sum_{i=1}^2 w_{i,t} R_{i,t} \\ &= w_1 * R_{1,t} + w_2 * R_{2,t} \end{split} \tag{3.1} \end{equation}\]

n = rets.shape[1]
weight = 1.0 / n
rets['Portfolio'] = (weight * rets).sum(axis=1)
rets.head()

3.1.2 Portfolio volatility

The other crucial aspect of a portfolio is its volatility.

\[\begin{equation} \begin{split} \sigma_{P,t} &= \sqrt{w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + 2 w_1 w_2 \sigma_1 \sigma_2 \rho_{1,2}} \\ &= \sum_{i=1}^2 \sum_{j=1}^2 w_i w_j \sigma_{i,j} \end{split} \tag{3.2} \end{equation}\]
